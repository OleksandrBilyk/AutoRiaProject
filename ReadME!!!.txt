Проект зроблено на основі лекцій по Django.
Проект докеризовано, тому запускати потрібно як звичайний докер контейнер docker compose up --build

Всі ендпойнти збережені у файлі AutoRiaProject.postman_collection.json
Також є документація swager - при запущеному контейнері ендпойнт http://localhost/api/doc
Для розміщення на AWS зробив zip архів. Але не встиг залити та перевірити роботу.

По завданню все виконано. Але є деякі моменти які б я переробив в майбутньому:

1. Зробив би frontend - бо в проекті тільки зачаток фронту, він не працюючий. З працюючим фронтом було б наглядніше.
2. Переробив би систему покупки premium для юзера. В проекті вона виконана як звичайний ендпойнт де треба ввести число.
Я знаю що існують спеціальні модулі для реалізації оплати на сайтах. В майбутньому замість простого енпойнта можна було б
використати платіжну систему або API для обробки платежів, щоб забезпечити безпеку та зручність для користувачів.
3. Зробив би чистішим свій код (наприклад переробив би в деяких view's отримання юзера через переопреділення методів
серіалайзера).
